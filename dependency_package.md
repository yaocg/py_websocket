

## 依赖包
```
pip3 install Django==3.0.2 --timeout=1500 --user
pip3 install channels==2.4.0 --timeout=1500 --user
pip3 install apscheduler --timeout=1500 --user
pip3 install channels_redis --timeout=1500 --user
pip3 install websocket-client --timeout=1500 --user
```

## 安装redis
```
sudo  apt-get install redis-server
```
